Property Binding

  • Property binding sets and element's property to the value of a Template Expression
  • The binding target is enclosed in []
  • The binding source is enclosed in ''

Event Binding

  • Event Binding listends for events and executes a template statement when the event occurs
  • The target event is enclosed in ()
  • The template statement is enclosed in ''

  • {{ listItemText }}



Two-Way Data Binding

  • Two-Way Data Binding displays a component class property and updates that property when the user makes a change
  • Two-Way Data Binding is achieved with the ngModel directive, exposed from the FormsModule
  • Use the [()] syntax (banana in a box) with the ngModel directive
  • The binding source is enclosed in ''
