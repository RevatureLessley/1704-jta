Data Binding


Interpolation

  • String Interpolation inserts interpolated string into the text within HTML Elements
  • String Interpolation uses double curly braces and NO QUOTES
  • {{interpolatedString}}
  • {{interpolatedNumber}}
  • {{interpolatedBoolean}}

Property Binding

  • Property Binding sets an element's property to the value of a Template Expression
  • The binding target is enclosed in square brackets []
  • The binding source is enclosed in single quotes ''

Event Binding

  • Event Binding listens for events and executes a template statement when the event occurs
  • The target event is enclosed in parenthesis ()
  • The template statement is enclosed in ''
  • {{ listItemText }}


Two-Way Data Binding

  • Two-Way Data Binding displays a component class property and updates that property when the user makes a change
  • Two-Way Data Binding is achieved with the ngModel directive, exposed from the FormsModule
  • Use the [()] syntax (banana in a box syntax) with the ngModel directive
  • The binding source is enclosed in single quotes ''
